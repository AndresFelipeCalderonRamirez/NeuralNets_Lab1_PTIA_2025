# NeuralNets_Lab1_PTIA_2025
This repository contains the implementation of Lab 1 for the course "Principles and Technologies of Artificial Intelligence (PTIA)" at the Colombian School of Engineering, semester 2025-2.

**Derivada función de costo: Entropia Cruzada:**

Para poder hacer la derivada de la función de Entropia Cruzada, entramos a WolframAplha y usamos el comando `- D[Sum[f[x] * Log[g[x]], {i, 1, n}], x]`
